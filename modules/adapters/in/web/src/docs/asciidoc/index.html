하우스 오브 옹알 유저 서비스 API 문서입니다.

유저(User) API

유저는 하우스 오브 옹알 서비스를 이용하는 사용자로, B2B(Business) 유저와 B2C(Customer) 유저로 구분됩니다.

닉네임 사용여부 확인 API

해당 닉네임이 사용중인지 확인합니다.

회원가입시 닉네임은 중복될 수 없습니다.

HTTP request

GET /users/nickname/leaf HTTP/1.1
Host: api.houseofongr.com

Path parameters

Table 1. /users/nickname/{nickname}
Parameter Optional Description

nickname

false

존재여부를 확인할 닉네임입니다.

Response body

{
  "exist" : true
}

Response fields

Path Type Description

exist

Boolean

해당 닉네임이 존재하는지 여부입니다.

비즈니스 유저 생성 API

비즈니스 유저를 생성합니다.

HTTP request

POST /users/business HTTP/1.1
Content-Type: application/json;charset=UTF-8
Content-Length: 155
Host: api.houseofongr.com

{
  "email" : "test@example.com",
  "password" : "temp2143@",
  "nickname" : "temp_user123",
  "termsOfUseConsent" : true,
  "personalInfoConsent" : true
}

Request fields

Path Type Optional Description

email

String

false

비즈니스 사용자의 이메일입니다.
* 인증 완료된 이메일만 가능합니다.

password

String

false

사용할 비밀번호입니다.

nickname

String

false

사용할 닉네임입니다.

termsOfUseConsent

Boolean

false

이용약관 동의 여부입니다.

personalInfoConsent

Boolean

false

개인정보 제공 동의여부입니다.

Response body

{
  "userID" : "01982da8-6e9c-7424-bfcd-8af146523454",
  "email" : "test@example.com",
  "nickname" : "temp_user123",
  "termsOfUseConsent" : true,
  "personalInfoConsent" : true
}

Response fields

Path Type Description

userID

String

임시 회원가입된 사용자의 ID입니다.(관리자 승인 후 활성화)

email

String

회원가입한 사용자의 이메일입니다.

nickname

String

회원가입한 사용자의 닉네임입니다.

termsOfUseConsent

Boolean

회원가입한 사용자의 이용약관 동의 여부입니다.

personalInfoConsent

Boolean

회원가입한 사용자의 개인정보 제공 동의여부입니다.

비즈니스 유저 승인 API

비즈니스 유저를 승인합니다.

Path parameters

Table 1. /users/{userID}/approve/{approve}
Parameter Optional Description

userID

false

승인할 비즈니스 회원의 ID입니다.

approve

false

해당 회원을 승인할지, 거부할지 여부입니다.

HTTP request

POST /users/01982da8-6e22-78e5-8155-d7577c9b8b50/approve/true HTTP/1.1
Host: api.houseofongr.com
Content-Type: application/x-www-form-urlencoded

Response body

{
  "userID" : "01982da8-6e22-78e5-8155-d7577c9b8b50",
  "nickname" : "leaf",
  "isApproved" : true
}

Response fields

Path Type Description

userID

String

회원가입된 사용자의 ID입니다.

nickname

String

회원가입된 사용자의 닉네임입니다.

isApproved

Boolean

승인 및 거절 여부입니다.

유저 정보 조회 API

특정 유저의 정보를 조회합니다.

Path parameters

Table 1. /users/{userID}/info
Parameter Optional Description

userID

false

조회할 사용자의 ID입니다.

HTTP request

GET /users/01982da8-6e87-7628-8580-7013f21b31b6/info HTTP/1.1
Host: api.houseofongr.com

Response body

{
  "id" : "01982da8-6e87-7628-8580-7013f21b31b6",
  "termsOfUseConsent" : true,
  "personalInfoConsent" : true,
  "email" : "test@example.com",
  "nickname" : "leaf",
  "userType" : "BUSINESS",
  "userStatus" : "ACTIVATE",
  "registeredAt" : 1753112669
}

Response fields

Path Type Description

id

String

조회된 사용자의 ID입니다.

termsOfUseConsent

Boolean

이용약관 동의여부입니다.

personalInfoConsent

Boolean

개인정보 활용 동의여부입니다.

email

String

이메일 주소입니다.

nickname

String

닉네임입니다.

userType

String

사용자 타입입니다.

userStatus

String

사용자 상태입니다.

registeredAt

Number

등록일자입니다.
